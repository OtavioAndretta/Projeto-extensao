🌌 Física Interativa: Laboratório Digital de Extensão
🚀 A Ciência na palma da mão
Este projeto nasceu como uma iniciativa de Extensão Universitária com um objetivo claro: democratizar o ensino de física. Através de simulações interativas e responsivas, transformamos fórmulas complexas em experiências visuais que rodam diretamente no navegador — do desktop ao celular.

O diferencial: Diferente de simuladores pesados, este laboratório é focado em performance e extrema responsividade, ajustando-se perfeitamente a telas sem perder a precisão dos cálculos.

🛠️ Experimentos Disponíveis
1. Queda Livre (Cinemática)
Explore como a gravidade e o arrasto do ar influenciam objetos em queda.

Modelos: Analítico (vácuo) e Numérico (com resistência do ar).

Ajustes em Tempo Real: Mude a gravidade (Terra, Lua, Marte) e a altura instantaneamente.

Exportação: Gere relatórios em .CSV dos dados simulados para análise em aula.

📱 Responsividade de Ponta
O grande desafio técnico superado foi a criação de um Canvas Dinâmico.

Auto-Scale: O simulador detecta o tamanho da tela e recalibra a escala de pixels por metro.

Touch Friendly: Controles desenhados para serem operados com o polegar.

DPI Aware: Gráficos nítidos em telas de alta densidade (OLED/AMOLED).

⚡ Tecnologias & Performance
Para garantir que o projeto seja acessível em escolas com internet limitada ou hardware básico, optamos pelo Minimalismo Tech:

HTML5 & CSS3: Layout moderno usando Grid e Flexbox (zero frameworks pesados).

Vanilla JavaScript: Toda a engine física foi escrita do zero para garantir máxima velocidade.

GitHub Pages: Hospedagem estática, segura e com carregamento instantâneo.
